Carmelo Pullara

Web Developer

Sono di Favara (AG). Mi occupo dello sviluppo di siti ed applicazioni web.

Sviluppo principalmente in Python, linguaggio di programmazione che reputo assolutamente il migliore tra quelli con cui ho avuto esperienza fino ad'ora. Nella maggior parte dei miei progetti Python utilizzo il web framework Django.

Oltre a Python, ho maturato molta esperienza con PHP, soprattutto grazie al framework Laravel che ritengo davvero validissimo.

Nonostante sia principalmente un back-end developer, mi occupo anche del fron-end delle applicazioni che realizzo, cercando di sviluppare sempre interfacce pulite e funzionali.

Inoltre, sono un grande appassionato di open source, soprattutto del mondo Linux.
